> 🌐 Mundo virtual 3D construido con Three.js y comunicación en tiempo real - Grupo 13

---

## 📋 Descripción

Un metaverso interactivo donde los usuarios pueden explorar un mundo 3D, personalizar su avatar, chatear en tiempo real y disfrutar de un entorno vivo con animales animados. Desarrollado como proyecto para la materia de Tecnologías Emergentes.

---

## 🎮 Características

- ✅ **Mundo 3D** con terreno, montañas, lago y vegetación
- ✅ **Avatares personalizables** (15 emojis diferentes)
- ✅ **Nombres visibles** sobre cada avatar
- ✅ **Multijugador en tiempo real** con Socket.io
- ✅ **Chat integrado** con mensajes instantáneos
- ✅ **Animales animados** (mariposas, pájaros y conejos)
- ✅ **Compatible** con PC y dispositivos móviles
- ✅ **Controles táctiles** para celulares

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Descripción |
|------------|-------------|
| **Three.js** | Motor 3D para renderizar el mundo virtual |
| **Socket.io** | Comunicación en tiempo real entre usuarios |
| **Node.js** | Servidor backend |
| **Express** | Framework para el servidor web |
| **HTML/CSS** | Interfaz de usuario y estilos |
| **JavaScript** | Lógica del cliente y servidor |

---

## 🚀 Instalación y ejecución

### Requisitos previos
- Node.js instalado en tu computadora
- Ngrok instalado (para compartir por internet)

### Pasos para ejecutar el proyecto

```bash
# 1. Instalar dependencias (solo la primera vez)
npm install express socket.io

# 2. Iniciar el servidor local
node server.js

# 3. Abrir el navegador en http://localhost:3000

# 4. Para compartir con compañeros (abrir otra terminal)
ngrok http 3000
